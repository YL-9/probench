# 嘤嘤的新平衡树

## 题目描述

给定一棵二叉树，二叉树的每个结点只有0或2个孩子。 

你需要对每个结点赋值一个正整数，使得每个结点的左右子树权值和相等。 

你需要返回所有结点的最小权值和对 $10^9+7$ 取模的结果。 

二叉树结点个数不超过$10^5$。 

示例1 

## 输入
    
    
    {0,0,0}

## 返回值
    
    
    3
