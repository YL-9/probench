# 两数之和

## 题目描述

$\,\,\,\,\,\,\,\,\,\,$对于给定的正整数 $z$ ，你需要寻找两个不同的正整数 $x$ 和 $y$ ，使得 $x+y=z$ 成立。 

$\,\,\,\,\,\,\,\,\,\,$如果不存在这样的 $x$ 和 $y$ ，你只需要输出 $\rm NO$ 。 

## 输入描述:
    
    
    $\,\,\,\,\,\,\,\,\,\,$在一行上输入一个整数 $z\ (1 \le z \le 2 \cdot 10^5)$ 。

## 输出描述:
    
    
    $\,\,\,\,\,\,\,\,\,\,$如果存在符合要求的答案，你需要先输出 $\rm YES$ ，随后在第二行上输出两个正整数 $x$ 和 $y$ 代表你的答案；否则，直接输出 $\rm NO$ 。

示例1 

## 输入
    
    
    2

## 输出
    
    
    NO

示例2 

## 输入
    
    
    3

## 输出
    
    
    YES
    1 2
