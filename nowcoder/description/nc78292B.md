# 小红的抛弃后缀

## 题目描述

小红拿到了一个正整数，她准备切掉一个后缀并抛弃，使得剩余部分是9的倍数。小红想知道有多少种不同的操作方案？

## 输入描述:
    
    
    一个正整数$x$  
    $1\leq x \leq 10^{100000}$

## 输出描述:
    
    
    一个整数，代表合法的方案数。

示例1 

## 输入
    
    
    1989

## 输出
    
    
    2

## 说明
    
    
    方案1：什么都不切（即切一个长度为0的后缀）。  
    方案2：切掉最后一个9（即切一个长度为1的后缀）。
