# 小红的同余

## 题目描述

给出一个奇数 $m$，请找出一个整数 $x$（$0 \leq x < m$），使得 $2 x \equiv 1 \pmod m$，也就是 $2 x$ 除以 $m$ 的余数是 $1$。你需要输出这个整数 $x$。  
  
可以证明，这样的整数 $x$ 存在且唯一。

## 输入描述:
    
    
    第一行一个整数 $m$，表示给出的奇数。  
    $3 \leq m \leq 10^9$

## 输出描述:
    
    
    输出一个整数 $x$（$0 \leq x < m$），满足 $2 x \equiv 1 \pmod m$。

示例1 

## 输入
    
    
    3

## 输出
    
    
    2

## 说明
    
    
    $2 \times 2 \equiv 1 \pmod 3$
