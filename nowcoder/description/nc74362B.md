# 小红的因子数

## 题目描述

小红拿到了一个正整数$x$，她想知道$x$有多少个不同的素因子，你能帮帮她吗？

## 输入描述:
    
    
    一个正整数$x$。  
    $1\leq x \leq 10^{13}$

## 输出描述:
    
    
    $x$不同的素因子的数量。

示例1 

## 输入
    
    
    1

## 输出
    
    
    0

示例2 

## 输入
    
    
    12

## 输出
    
    
    2
