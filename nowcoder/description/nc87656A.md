# 小红的字符串

## 题目描述

$\,\,\,\,\,\,\,\,\,\,$小红拿到了一个长度为 $3$ 且仅由小写字母组成的字符串，她每次操作可以修改任意一个字符，小红希望最终所有字符都相同。 

$\,\,\,\,\,\,\,\,\,\,$你能帮小红求出最小的操作次数吗？ 

## 输入描述:
    
    
    $\,\,\,\,\,\,\,\,\,\,$第一行输入一个长度为 $3$ 且仅由小写字母组成的字符串 $s$。

## 输出描述:
    
    
    $\,\,\,\,\,\,\,\,\,\,$在一行上输出一个整数，表示最小的操作次数。

示例1 

## 输入
    
    
    aba

## 输出
    
    
    1

## 说明
    
    
    $\,\,\,\,\,\,\,\,\,\,$操作 $1$ 次，将第二个字符修改为 '$\tt a$' 即可。

示例2 

## 输入
    
    
    uuu

## 输出
    
    
    0

时间限制：C/C++/Rust/Pascal 1秒，其他语言2秒
