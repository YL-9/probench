# 交换数字

## 题目描述

Baijiaohu有两个长度均为 $n$ 且不包含前导零的数字 $a,b$ ，现在他可以对这两个数字进行任意次操作：   
  


  1. 选择一个整数 $1 \le i \le n$ ,并交换 $a,b$ 的第 $i$ 位 。 

  
请输出任意次操作后 $a \times b$ 的最小值，由于答案可能很大，请对 $998244353$ 取模。  


## 输入描述:
    
    
    第一行输入一个数字 $n$ 代表两个数字的长度   
    第二到三行输入两个字符串 $a,b$   
    $1 \le n \le 2\times 10^5$  
    

## 输出描述:
    
    
    输出一个 $ans$ 表示最后的答案   
    请对 $998244353$ 取模  
    

示例1 

## 输入
    
    
    3
    159
    586

## 输出
    
    
    91884

示例2 

## 输入
    
    
    10
    1578959751
    1786548221

## 输出
    
    
    410002876
