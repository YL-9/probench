# 唐龙守则

## 题目描述

众所周知，牛客小白月赛q群有一条特殊规则：  
不能连续发三张及以上的奶龙表情包。  
但是，发多了也没关系，好心的管理员会对表情包序列进行处理。  
只要每三张撤回一张，炸鸡块就没办法禁言了！  
给定 表情包序列 的长度 $n$ ，请问管理员需要撤回几张表情包。  


## 输入描述:
    
    
    第一行有一个正整数 $n\ (\ 1 \leq n \leq 10^5\ )$ 。

## 输出描述:
    
    
    输出一个整数，代表需要撤回的表情包的数量。

示例1 

## 输入
    
    
    7

## 输出
    
    
    2
