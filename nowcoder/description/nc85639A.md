# 嘤嘤不想做计几喵

## 题目描述

如果一场比赛的题目总数为 $a$ ，计算几何题数量为 $b$ 。那么嘤嘤对这场比赛的评价为：$(a-b)-b \times 10$ 。  
  
现在给出一场比赛的题目总数和计算几何题数量，你需要算出嘤嘤对这场比赛的评价喵~。

## 输入描述:
    
    
    给定两个整数 $a(1 \leq a \leq 10^9),b(0 \leq b \leq a)$ ，表示题目总数、计算几何题数量。

## 输出描述:
    
    
    输出一个整数表示答案。

示例1 

## 输入
    
    
    6 1

## 输出
    
    
    -5

## 说明
    
    
    叠甲：本场比赛只有本题是嘤嘤出的。


## 时空限制

时间限制：C/C++/Rust/Pascal 1秒，其他语言2秒
