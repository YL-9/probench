# S 老师的公式

## 题目描述

S 老师丢给你了一个简单的数学问题：

求 。

请你求出答案。

## 输入描述:
    
    
    **输入格式**
    
     一行一个整数 $n$ $(1\leq n \leq 10^6)$。

## 输出描述:
    
    
    一行一个整数表示答案。

示例1 

## 输入
    
    
    3

## 输出
    
    
    6

## 说明
    
    
    $\gcd(1+2+3,1\cdot 2\cdot 3) = \gcd(6,6)=6$
