# 小红的01串（一）

## 题目描述

小红拿到了一个01串，她想知道这个01串有多少对相邻的字符满足，这两个字符是不同的？ 

## 输入描述:
    
    
    一个长度不超过100的字符串，仅由'0'和'1'组成。

## 输出描述:
    
    
    输出相邻的不同字符的对儿数。

示例1 

## 输入
    
    
    01011

## 输出
    
    
    3

## 说明
    
    
    有3对相邻的字符不相同。
