# 小红的二进制删数字

## 题目描述

小红拿到了一个二进制字符串 $s$，她可以删掉其中的一些字符，使得最终该字符串为一个2的幂（即可以表示为 $2^k$ 形式的数）。小红想知道，自己最少删几个字符可以达成？请你编写一个函数返回这个答案。  
$1\leq len(s) \leq 10^5$

示例1 

## 输入
    
    
    "111"

## 返回值
    
    
    2
    

## 说明
    
    
    删掉前两个 '1'，字符串变成"1"，$1=2^0$ 为2的幂。

示例2 

## 输入
    
    
    "1010"

## 返回值
    
    
    1
    

## 说明
    
    
    删掉第三个字符 '1'，字符串变成"100"，代表的数是 $4=2^2$
