# 小红的子数组排列判断

## 题目描述

小红拿到了一个数组，她想知道有多少个长度为$k$的连续子数组是排列？  
  
定义排列为：1到$k$每个元素都恰好出现了1次。

## 输入描述:
    
    
    第一行输入两个正整数$n,k$，代表小红拿到的数组大小和连续子数组的大小。  
    第二行输入$n$个正整数$a_i$，代表数组中的元素。  
    $1\leq k \leq n \leq 10^5$  
    $1\leq a_i \leq 10^5$

## 输出描述:
    
    
    一个整数，代表长度为$k$的连续子数组是排列的数量。

示例1 

## 输入
    
    
    5 2
    1 2 1 1 2

## 输出
    
    
    3

## 说明
    
    
    有3个长度为2的连续子数组是排列：两个[1,2]和一个[2,1]
