# 小红平分糖果

## 题目描述

小红拿到了$n$个糖果，她准备和小紫平分这些糖果，也就是说她们拿到的糖果数量必须相同。请你计算她们各自分到多少糖果。

## 输入描述:
    
    
    一个正整数$n$。  
    $1\leq n \leq 100$

## 输出描述:
    
    
    如果无法平分，请输出-1。  
    否则输出两个正整数$x,y$，用空格隔开，代表小红和小紫拿到的糖果。

示例1 

## 输入
    
    
    3

## 输出
    
    
    -1

## 说明
    
    
    无论小红分到几颗糖果，小紫的糖果数量都无法和小红的相同。

示例2 

## 输入
    
    
    6

## 输出
    
    
    3 3

时间限制：C/C++/Rust/Pascal 1秒，其他语言2秒
