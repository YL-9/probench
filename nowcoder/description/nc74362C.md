# 小红的字符串中值

## 题目描述

小红定义一个长度为奇数的字符串的“中值”为中间那个字符。例如"kou"的中值是'o'。  
现在小红拿到了一个字符串，她想知道某个字符是多少个子串的中值。你能帮帮她吗？

## 输入描述:
    
    
    第一行输入一个正整数$n$和一个英文小写字符$chr$。代表字符串长度和询问的字符。  
    第二行输入一个长度为$n$的、仅由小写字母组成的字符串。  
    $1\leq n \leq 10^5$

## 输出描述:
    
    
    一个整数，代表中值为$chr$的连续子串数量。

示例1 

## 输入
    
    
    4 b
    abcb

## 输出
    
    
    3

## 说明
    
    
    有两个"b"字符串和一个"abc"字符串的中值都是'b'。
