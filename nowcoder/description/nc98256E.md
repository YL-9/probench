# 小红的01串（五）

## 题目描述

小红拿到了一个01串，但其中有一些被'?'字符代替。  
小红希望将'?'替换成'0'或者'1'字符，使得该串表示的正整数是13的倍数。你能帮小红求出有多少种方案吗？  
由于答案过大，请对$10^9+7$取模

## 输入描述:
    
    
    一行仅由'0'、'1'、'?'三种字符组成的字符串。保证第一个字符是'1'。  
    字符串长度不超过200000。

## 输出描述:
    
    
    一个整数，代表最终合法方案数对$10^9+7$取模的答案。

示例1 

## 输入
    
    
    10?1

## 输出
    
    
    1

## 说明
    
    
    只有1001这一个合法解。
